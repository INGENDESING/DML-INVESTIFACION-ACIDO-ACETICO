# Ácido Acético — Producción Industrial a Escala Mundial

Presentación web interactiva con investigación exhaustiva sobre las tecnologías de producción de ácido acético a escala industrial.

## 🧪 Contenido

- **6 procesos industriales** analizados: Cativa, AO Plus, Monsanto, Wacker, LPO, Fermentación
- **Comparativa técnica maestra** con tabla interactiva y gráfico radar
- **Datos de mercado global** con gráficos interactivos (Recharts)
- **15 insights estratégicos** del cruce de 12 dimensiones de investigación
- **Timeline de tendencias** 2025–2035

## 🛠 Stack Tecnológico

- React 19 + TypeScript 5.9
- Vite 7
- Tailwind CSS 3.4 + shadcn/ui
- Recharts (visualizaciones)

## 🚀 Desarrollo Local

```bash
npm install
npm run dev     # http://localhost:3000
```

## 📦 Build de Producción

```bash
npm run build   # Genera /dist
npm run preview # Preview local del build
```

## 🌐 Deploy en Render (Static Site)

| Campo | Valor |
|-------|-------|
| **Build Command** | `npm install && npm run build` |
| **Publish Directory** | `dist` |

## 📊 Fuentes de Datos

Investigación basada en 22 documentos académicos, patentes USPTO, informes SEC, datos Nexant/ChemSystems e informes de mercado. Datos de precios: Q4 2025.
